QA Project

Stack Requirements
--------
* Selenium using java
* Maven
* TestNG

### How to setup. 
1- make sure that you have the following 
- JDK 
- ChromeDriver 
- Firefox Driver
2- IDE installed in your machine (inteliJ or eclipse)
